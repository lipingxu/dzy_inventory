# 图书资产管理系统 (Inventory Management System)

## 项目简介
该系统已从单纯的价格追踪器升级为专业的**图书资产财务管理系统**。它不仅能记录购入/售出记录，还能通过多抓鱼行情实时计算资产总额、浮动盈亏，并提供可视化的资产走势图。

## 核心功能更新 (2026-05-28)

### 1. 财务级报表看板
*   **多维指标**: 自动计算“总投入成本”、“当前总估值”、“总浮动盈亏”及“已实现利润”。
*   **可视化趋势**: 集成 ECharts，展示近 7 次同步的**总浮动盈亏走势图**，资产波动一目了然。
*   **智能持仓**: 自动识别“真正持仓”（已填购入价的书籍）并进行专项财务统计。

### 2. 深度交互体验
*   **全表搜索**: 支持通过书名、ISBN 实时过滤表格内容。
*   **动态排序**: 点击任意表头（购入价、盈亏、趋势等）即可进行升序/降序排列，并伴有视觉箭头反馈。
*   **极致排版**: 日期列精简为 `MM-DD` 格式，大幅节省横向空间。

### 3. 数据安全与兼容性
*   **ISBN 自动保护**: 脚本在保存 CSV 时会自动为 ISBN 添加单引号前缀 (`'`)。
*   **Excel 完美兼容**: 彻底解决 Excel 打开 CSV 时 ISBN 变成科学计数法或丢失精度的问题。
*   **智能修复**: 同步过程中会通过“书名匹配”自动修复之前损坏的 ISBN。

## 核心文件说明
*   `sync_data.py`: **唯一运行脚本**。整合了剪贴板读取、CSV 数据库更新和 HTML 报表生成。
*   `inventory.csv`: **核心数据库**。存储所有书籍的元数据、状态、价格历史及保护后的 ISBN。
*   `report.html`: **可视化前端**。包含看板、图表、交互式表格。

## 快速操作指南

### 1. 更新行情与同步 (核心)
1.  **多抓鱼操作**：在 Chrome 开发者工具 `Network` 中，右键点击 `inquiry-books` -> `Copy` -> `Copy response`。
2.  **执行同步**：运行 `python3 sync_data.py`。
3.  **查看结果**：双击打开 `report.html`。

### 2. 维护购入价格
直接用 Excel 或文本编辑器打开 `inventory.csv`，在 `购入价格` 列填入您的买入成本并保存。程序下次运行时会自动计算该书的盈亏并更新报表。

## 报表视觉逻辑
*   **红色/向上箭头 (↑)**：表示价格上涨、盈利。
*   **绿色/向下箭头 (↓)**：表示价格下跌、亏损。
*   **灰色行**：表示多抓鱼当前不收的书籍。
*   **单引号保护**：CSV 中的 ISBN 前缀 `'` 在 Excel 中会自动隐藏，不影响使用。
